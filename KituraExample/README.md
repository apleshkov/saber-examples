# KituraExample

Contains:
- in-memory user storage
- Basic HTTP auth

Routes:
- `/` to list stored users
- `/logout` to log out

Predefined users:
- **admin** / 1234: can see user list
- **john** / 1234: can't see user list

Invoke `swift run` to start and open [localhost:8080](http://localhost:8080)
